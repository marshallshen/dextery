# Dexter

This is a Gemified wrapper of [Jekyll Boostrap](http://jekyllbootstrap.com/)

## Installation

Add this line to your application's Gemfile:

    gem 'dexter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dexter
